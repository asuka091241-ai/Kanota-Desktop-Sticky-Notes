关于 Kanota

Kanota
版本v1.0.3
作者CaffYooO
仓库github.com/asuka091241-ai/Kanota
邮箱13257035176@163.com

Kanota 是一款轻量级桌面看板便签应用，融合了看板卡片管理与桌面便签两大功能。支持拖拽流转卡片状态、专注模式、番茄钟计时、多主题切换等特性，帮助你在本地高效管理任务与灵感。

所有数据存储于本地，完全离线工作，无需任何网络连接。通过 GitHub Release 进行版本更新，你可以随时检查并下载最新版本。

行为

系统登录时自动运行 Kanota
新建便签拖到桌面时自动折叠为标题栏
便签中显示截止时间区域
专注模式下同样显示截止时间
专注模式下显示折叠的子任务区域（左下角箭头展开）
桌面便签左下角显示卡片创建时间
点击关闭时直接执行上次选择，不再弹窗询问
